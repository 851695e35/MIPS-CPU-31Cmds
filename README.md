# MIPS-CPU-31Cmds

31指令，单指令周期的哈佛架构 MIPS CPU，学校的计算机组成原理课程设计作业之一。

应该是通过了所有指令的测试了，但有一些Mars初始寄存器没有归零导致数据出错的指令没有处理，我懒，所以那些就没有考虑了。

`srcs`文件夹下是源代码和testbench代码，模拟时间请自行扩展（我扩展到了8000ns），`test`文件夹下是原始的指令和编译修改之后的coe文件，具体可以看情况使用。

有任何问题您可以在Issue里提出，注意这个仓库的目的仅仅是为不知从何下手的同学们指一条可能可以尝试的路，而不是说给大家抄作业的机会，细节部分还是要自己去仔细考虑设计斟酌的。

完整的项目工程会在Release里发布，可以去下载压缩包文件。请注意自行调整相关coe文件的位置。
